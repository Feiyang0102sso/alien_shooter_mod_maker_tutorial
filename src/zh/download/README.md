# 属性测试页面内容

当你保存并查看此页面时：

1. **顶部区域**：你应该能看到作者、日期、分类、标签，以及你关心的 **阅读时间** 和 **字数统计**。
2. **侧边栏**：会显示 `简短标题` 而不是那个长标题。
3. **底部**：会显示版权声明和评论区。

## 什么是阅读时间 (ReadingTime)？
这是一个自动计算的属性。主题会根据你正文的字数，按照每分钟约 300-500 字的速度预估一个时间。你不需要手动去写 `readingTime: 5min`，只要开启了 `pageInfo` 里的配置，它就会自动出现。

## 什么是类别 (Category) 与 标签 (Tag)？
- **Category**：通常用于大类，比如“下载”、“技术”。
- **Tag**：用于细分描述，比如“Windows”、“免费软件”。
